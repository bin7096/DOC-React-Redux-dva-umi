# `有状态组件`和`无状态组件`
> React中既有传统的有状态组件（类组件），也有新增的无状态组件（函数式组件）
## `有状态组件`和`无状态组件`的区别
* 有状态组件
  1. 以`class（类）`的形式创建，继承于`React.Component`。
  2. 拥有自己的`state`和`props`，props使用`this.props`接收。
  3. `会`触发生命周期的一系列钩子函数。
* 无状态组件
  1. 以`function函数`的形式创建，返回一个`JSX语法的组件`。
  2. 函数参数接收`props`，没有自己的`state`。
  3. `不会`触发生命周期的一系列钩子函数。
